# Frontend - Lead Management System

React + Vite frontend for the Lead Management assignment.

## Pages

- Login
- Register
- Dashboard
- Leads

## Features

- JWT auth integration with backend
- Protected routes
- Lead list with search and status filter
- Add, edit, delete lead
- Status update workflow (New, Contacted, Converted)
- File upload support in lead form

## Commands

```bash
npm install
npm run dev
```

Set environment variable in `.env`:

VITE_API_BASE_URL=http://localhost:5000/api
